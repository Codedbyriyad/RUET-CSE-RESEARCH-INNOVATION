# RUET CSE — Research Section (Frontend Only)

> React + Vite + Tailwind CSS  
> কোনো backend বা database লাগবে না ✅

---

## ⚡ Quick Start

```bash
# 1. Dependencies install করো
npm install

# 2. Dev server চালু করো
npm run dev
```

→ Live Demo: https://researchfontend.netlify.app/**

---

## 🗂 Project Structure

```
src/
├── App.jsx               ← Root component, tab state
├── main.jsx              ← Entry point
├── index.css             ← Global styles + CSS variables
├── utils/
│   └── data.js           ← সব static data এখানে
└── components/
    ├── Navbar.jsx
    ├── Hero.jsx
    ├── Tabs.jsx
    ├── ResearchAreas.jsx
    ├── Publications.jsx   ← Filter by type/year
    └── Projects.jsx
```

## ✏️ Data পরিবর্তন করতে চাইলে

`src/utils/data.js` ফাইল খোলো — সব research areas, publications, projects এখানে আছে। সরাসরি edit করো।
